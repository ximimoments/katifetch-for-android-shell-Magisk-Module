## Katifetch is now avaible on Magisk!

Katifetch is on magisk/ksu module where show your android info, like neofetch and fastfetch, but for android, is faster, and is the first neofetch alt on make a fetch for android shell, like, only for the shell of android

## For the install, uninstall and update, you have to need

- Root
- Magisk or ksu (kernel su)
- For the terminal use termux with command "su" or for pc "adb shell" and then "su"
- the zip file
